# &nbsp;&nbsp;&nbsp;     JavaScript&TypeScript学习报告  
JavaScript 的主要作用是为网页增加交互性和动态性。是一种客户端脚本语言，可以在网页上运行，并与用户进行交互。  

**基本语法**  
**①大小写敏感**  
例如：变量 myVariable 与 MyVariable 是不同的  

**②标识符**  
即指变量、函数、属性的名字，或者函数的参数。  
规则：第一个字符只能是字母、下划线（ _ ）或美元符号（ $ ）之一；  
其他字母可以是字母、数字、下划线或美元符号  
按照惯例，标识符采用**驼峰**大小写格式，第一个字母小写，剩下的每个单词的首字母大写，如**myProject**  

**③注释**  
//(双斜杠)后的内容将会被浏览器忽略。（单行注释）  

/**/  （多行注释）  

**④语句**  
每个语句后可以添加分号，也可以不添加  

**⑤变量**  
js的变量是松散类型的，不像c、c++等语言是强类型的，js在声明一个变量时，不必指明这个变量是什么类型的，只需要在声明变量时，使用var关键字，后加变量名，即可声明一个变量。  

**注意：** 使用var声明变量时，这个变量的作用域只在定义该变量的作用域生效，例如在一个函数体内使用了var定义变量，该变量只在该函数内生效，函数退出后，该变量也随即被销毁。  

去掉var，可以将该变量变为全局变量，不过不建议将变量声明为全局变量，因为维护全局变量的代价太大。  

**⑥操作符**  
1.**一元操作符**  
例如：++、--  
2.**布尔操作符**  
该操作符得到的结果为true/false两种情况。  
**除下列值为假外其余皆为真**： false、null、undefined、''、0、NaN  

**&&和||都属于 短路操作！**  
短路操作： 执行完一处得到了结果，就不必执行另一处的代码， &&操作符，前面的判断为true或false，后面的代码不执行，||操作符，前面的true，后面的代码不执行。  

3.**算术运算符**  
`+ - * /`  
var result = 5 + '5';  
结果为 result='55'  
+运算符会把前面的数字5当成字符'5'，和后面的字符'5'连接起来，变成了字符串'55'  

4.**关系操作符**  
`<> <= >= == === != !==`  
`==`值相等,  
var x = 5;  
console.log(x == '5');打印为true  
变量x会转换为'5'与之相比较，所以会打印true  
`===`全等（值和类型都相等）  
建议使用全等比较符，不容易出错  
console.log(x === '5');打印false  
因为变量x的类型是number，'5'的类型是string.  


**5.语句**  
`if do-while while for for-in for-of break continue switch`      
编写一个语句块时，需添加`{}`  

**传统遍历**  
`for(var i=0;i<array.length;i++)`  
如果需要提升遍历的性能，可以将array.length替换成一个已经赋值的变量,这样就不会每次都去计算array.length的长度  

```
for(var c in array)
{
  console.log(array[c]);
}
```  
for-in，是针对下标的

```
for(var c of array){
  console.log(c);
}
```  
for-of,是针对数组元素的  


**6.函数**  
格式: function 函数名(变量1，变量2){}  
函数不一定具有返回值，如果函数需返回值的时候，在函数体内用`return`返回  

在调用函数的过程中，可以向函数传递多个参数，也不在乎传递进来的参数的数据类型是什么  
这种规则，可能会造成一些不可察觉的错误。  

**7.对象Object**  
通常将属性和方法封装在对象中  

**两种方式创建对象**  
①  
```
    // new
    var car=new Object  //(创建一个空对象)
    car.brand='BMW'   //设置对象的brand属性
    car.run=function (){        //设置了对象的run()方法
        console.log(car is running···)
    }
```  
②  
```
    //字面量
    var car={
      brand:'BMW',
      run:function(){
        console.log(car is running···)
      }
    }
```  
以上两种方式创建很多对象的话，会产生大量重复的代码，
于是人们引入了设计模式之工厂模式  
（设计模式：解决一个问题的有很多种，但经过人们的总结，解决这个问题的最优的解决方式，就是设计模式）  

例如:  
```
  function createCar(brand){
    var car=new Object()
    car.brand=brand
    car.run=function(){
      console.log(car is running···)
    }

    return car  
  }
```  
var car1=createCar('BMW')  
var car2=createCar('Rolls-Royce')  
运用了工厂模式创建了两个对象，提高了代码的复用性。  

**8.数组**  
可以使用数组字面量或数组构造函数来创建数组。  

`var fruits = ['apple', 'banana', 'orange'];`  
使用数组字面量（方括号 []）创建数组  

`var numbers = new Array(1, 2, 3, 4, 5);`  
使用数组构造函数创建数组  


```
let fruits = ['apple', 'banana', 'orange'];

console.log(fruits[0]); // 输出 'apple'

fruits[1] = 'grape'; // 修改第二个元素

console.log(fruits); // 输出 ['apple', 'grape', 'orange']
```  
可以使用索引来访问和修改数组中的元素  

**数组的常用方法**  
push()：向数组末尾添加一个或多个元素。  
pop()：移除并返回数组的最后一个元素。  
shift()：移除并返回数组的第一个元素。  
unshift()：向数组开头添加一个或多个元素。  
length：获取数组的长度（元素个数）。  

**数组的两种分片方法**  
**①slice(startIndex, endIndex)**  
slice() 方法可以从原数组中提取一部分元素，返回一个新的数组。它接受两个参数：起始索引和结束索引（不包括在内），表示要提取的子数组的范围。如果省略结束索引，则会提取从起始索引到数组末尾的所有元素。  

**②splice(startIndex, deleteCount, item1, item2, ...)**  
splice() 方法用于修改数组，可以删除、替换或插入元素。它接受三个或多个参数：起始索引、要删除的元素数量（deleteCount）和要插入的新元素（如果有）。返回值是被删除的元素组成的数组。  

**链式语法**  
链式语法：一种通过在方法调用之间使用点号来连接多个方法的编码风格。  

如何使用链式语法？  
确保每个方法调用返回一个对象，以便可以继续在该对象上进行其他方法调用。  

**输出**  
**①console.log()**  
最常用的输出方法之一，它可以将文本、变量和表达式的值输出到浏览器的控制台，供调试和日志记录使用。  
**②alert()**  
可以在浏览器中创建一个简单的弹窗，用于向用户显示文本消息。  

#     <center>TypeScript  
TypeScript是JavaScript的超集，扩展了JavaScript的功能，比js的功能更强大  

**1.let、 const、var**  
①let  
用于声明块级作用域的变量，可以在同一作用域内重新赋值  
②const  
用于声明常量，一旦赋值后不可修改  
③var  
如果在函数外部或没有函数包裹的代码块内使用 var 声明变量，那么该变量将具有全局作用域  

总结：var是早期js的变量声明方式，但在ts中，更推荐let和const的使用，因为它们提供了更好的作用域控制和更严格的变量声明行为，可以减少代码的维护量。  

**2.解构**  
①数组解构  
将数组中的值按顺序解构到单独的变量中  
例子:  
```
const numbers: number[] = [1, 2];

// 数组解构
const [first, second] = numbers;

console.log(first);  // 输出: 1
console.log(second); // 输出: 2
```  
②对象解构  
将对象中的属性值解构到单独的变量中  
例子：  
```
const person = {
  name: 'Alice',
  age: 30,
  city: 'New York'
};

// 对象解构
const { name, age, city } = person;

console.log(name); // 输出: Alice
console.log(age);  // 输出: 30
console.log(city); // 输出: New York
```  

**3.函数**  
不同于js的函数，ts提供了声明函数的参数类型和返回值类型，以提供更强的类型检查，从而减少维护成本  

①可选参数  
可选参数可以在函数调用时**省略**某些参数，使用 '?' 标记在参数名后面,且**必须放在必要参数后**。  
②默认参数  
可以在函数定义时给参数提供默认值，**不必在必要参数后**。  
③剩余参数  
可以函数接受任意数量的参数，使用' ... '前缀标记参数名，  
④箭头函数  
简化了函数的定义  
例子：  
```
  let multiply = (a: number, b: number): number => a * b;
```  
若函数体有多行，用{}括起来.  
**4.类class**  
类是属性（有些什么）和函数（能干什么）的集合,是生成对象（Object）或类实例的模板.  
①类的定义和使用  
类名应使用大驼峰命名法(SuperMan);  
使用构造函数来初始化类的属性;  
②类的属性和函数的访问权限  
ts提供三种属性访问修饰符：public、private 和 protected，用于控制类成员的可访问性。  
public：公开访问，可以在类内部和外部访问；  
private：私有访问，只能在类内部访问；  
protected：受保护访问，可以在类内部和子类中访问
③存取器-getter、setter  
存取器是用于定义属性的读取和修改方式的一种语法。  
'get'读取，'set'修改  
④静态属性  
是属于类本身而不是类的实例的属性，可直接使用而不需要实例化。  
⑤继承  
使用关键字 'extends' 来实现，以从现有的类中继承属性和方法  
子类的构造函数：  
子类在构造函数中必须调用父类的构造函数，将父类属性初始化，使用'super'关键字来调用父类构造器。  
**5.模块**  
模块可以包含变量、函数、类和其他模块。  
（1）导入  
使用'import'关键字导入其他模块的成员  
(2) 导出  
使用'export'关键字将模块的成员（变量、函数、类等）导出
